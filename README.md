# Runescape-Autouquest
Program to allow completion of Runescape quests automatically.
This program will provide the user with different options of quests and skills to train.
  List of Quests:
    1) Cooks assistant
    2) Restless ghost
    3) Sheep Sheerer
  List of Skills:
    1) Woodcutting
    2) Firemaking
    3) Fishing
    4) Cooking
